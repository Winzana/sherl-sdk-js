## Get user position

To retrieve user position

```ts
const me = await sherl
  .person()
  .position({ longitude: 'your_longitude', latitude: 'your_latitude' });
```

Return object

```ts
{
  id: string;
  country: string;
  locality: string;
  region: string;
  postalCode: string;
  streetAddress: string;
  latitude: string;
  longitude: string;
}
```

## Get user config

To retrieve pro user config

```ts
const me = await sherl.person().config();
```

Return object

```ts
{
  id: string;
  code: string;
  value: any;
  consumer: string;
  position: number;
  appliedTo?: string;
  isPublic?: boolean;
}
```
