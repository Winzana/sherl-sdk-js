## Import SDK

```ts
// With require
const Sherl = require('@sherl/sdk');
// OR import
import * as Sherl from '@sherl/sdk';
// OR
import { domain } from '@sherl/sdk'; // we will use this import for the rest of the documentation
```
where "domain" is the sherl product (person, auth, ...)
